                                                               iana-if-type YANG Module

   Created
           2014-01-27

   Last Updated
           2015-06-12

   Description

 Interface types must not be directly added to the iana-if-type YANG
 module. They must instead be added to the "ifType definitions" registry
 at [IANA registry smi-numbers].
