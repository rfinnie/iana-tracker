   This registration tracks a change made to ISO 639-3, correcting the
   reference name for code element 'dgt' from "Ndrag'ngith" to
   "Ndra'ngith". The correct spelling was stated in the original change
   request. This correction originally appeared in ISO 639-3 data files
   dated 2013-05-20, and was not listed in the Summary of Outcomes
   document for any change series.

   The original change request to add this ISO 639-3 code element is at:
   http://www-01.sil.org/iso639-3/cr_files/2011-117.pdf
