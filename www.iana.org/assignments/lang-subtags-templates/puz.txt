This registration tracks a change made to ISO 639-3 effective
2014-02-03, merging 'puz' for "Purum Naga" into 'pub' ("Purum").

This change was not listed directly in the Series 2013 Summary of
Outcomes document. It was described in a document located at
http://www-01.sil.org/iso639-3/cr_files/PastComments/CR_Comments_2013-008.pdf
which describes the decision to retire code element [puz] and change the
denotation of code element [pub].
