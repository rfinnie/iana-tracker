This registration tracks a change made to ISO 639-3 effective
2016-01-15, adding the additional name "Sekak" for the existing code
element 'lce'. Both the existing primary language subtag and the
existing extended language subtag were modified. This name change does
not affect the existing macrolanguage association with Malay ('ms').

For more information on the ISO 639-3 change, refer to:
http://www-01.sil.org/iso639-3/cr_files/639-3_ChangeRequests_2015_Summary.pdf
