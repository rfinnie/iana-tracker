This registration tracks a change made to ISO 639-3, adding the code
element 'yhs' for Yan-nhaŋu Sign Language. This change originally
appeared in ISO 639-3 data files dated 2015-03-30, and was not listed in
the Summary of Outcomes document for any change series. Both a primary
language subtag and an extended language subtag were added for this code
element.
