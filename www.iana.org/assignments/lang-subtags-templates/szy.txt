This registration tracks a change made to ISO 639-3 effective
2019-01-25, splitting the code element 'ais' for Nataoran Amis into the
existing code element 'ami' (Amis) and a new code element 'szy'
(Sakizaya).

For more information on the ISO 639-3 change, refer to
https://iso639-3.sil.org/sites/iso639-3/files/reports/639-3_ChangeRequests_2018_Summary.pdf
