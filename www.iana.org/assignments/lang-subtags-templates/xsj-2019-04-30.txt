This registration tracks a change made to ISO 639-3 effective
2019-01-25, reactivating (removing the "Deprecated" status for) the code
element [xsj] for Subi. This code element was originally withdrawn from
ISO 639-3 on 2015-01-12. A Comments field is added to reflect the fact
that Subi was encoded as [suj] between 2015 and 2019.

For more information on the most recent ISO 639-3 change, refer to:
https://iso639-3.sil.org/sites/iso639-3/files/reports/639-3_ChangeRequests_2018_Summary.pdf
