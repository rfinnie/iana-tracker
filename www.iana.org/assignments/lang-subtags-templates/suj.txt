This registration tracks a change made to ISO 639-3 effective
2019-01-25, reactivating (removing the "Deprecated" status for) the code
element [xsj] for Subi. No change was made to the ISO 639-3 record for
[suj], but a Comments field is added to the Language Subtag Registry to
reflect the fact that Subi was encoded as [suj] between 2015 and 2019.

For more information on the ISO 639-3 change, refer to:
https://iso639-3.sil.org/sites/iso639-3/files/reports/639-3_ChangeRequests_2018_Summary.pdf
