This registration tracks a change made to ISO 639-3 effective
2015-01-12, retiring the code element 'yds' for Yiddish Sign Language as
non-existent. No Preferred-Value is indicated. This change affects both
the primary and extended language subtags for this language.

For more information on the ISO 639-3 change, refer to:
http://www-01.sil.org/iso639-3/cr_files/2014-010.pdf

This change was approved but not listed in the Series 2014 Summary of
Outcomes document.
