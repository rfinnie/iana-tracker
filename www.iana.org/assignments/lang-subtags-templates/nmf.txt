   This registration tracks a change made to ISO 639-3 effective
   2012-02-06, splitting the new code element 'ntx' for "Tangkhul Naga
   (Myanmar)" from the existing code element 'nmf' for "Tangkhul Naga"
   and renaming the latter to "Tangkhul Naga (India)".
