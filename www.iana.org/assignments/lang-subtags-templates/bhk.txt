   This registration tracks a change made to ISO 639-3 effective
   2010-01-20, splitting the code element 'bhk' for Albay Bicolano into
   four new code elements: 'fbl' (West Albay Bikol), 'lbl' (Libon
   Bikol), 'rbl' (Miraya Bikol), and 'ubl' (Buhi'non Bikol).
