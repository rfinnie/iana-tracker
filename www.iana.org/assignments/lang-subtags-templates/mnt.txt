   This registration tracks a change made to ISO 639-3 effective
   2013-01-23, splitting the code element 'mnt' for Maykulan into four
   new code elements: 'xyk' (Mayi-Kulan), 'wnn' (Wunumara), 'xyt'
   (Mayi-Thakurti), and 'xyj' (Mayi-Yapi).

   For more information on the ISO 639-3 change, refer to:
