   This registration tracks a change made to ISO 639-3 effective
   2012-02-06, splitting the code element 'nln' for Durango Nahuatl into
   two new code elements, 'azd' (Eastern Durango Nahuatl) and 'azn'
   (Western Durango Nahuatl).

   For more information on the ISO 639-3 change, refer to
   http://www.sil.org/iso639-3/cr_files/639-3_ChangeRequests_2011_Summary.pdf .
