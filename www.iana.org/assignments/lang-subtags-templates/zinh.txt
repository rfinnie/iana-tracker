3. Record Requested:

Type: script
Subtag: Zinh
Description: Code for inherited script
Added: 2009-04-03
Comments: Not intended for use as a language subtag

4. Intended meaning of the subtag:

In the Unicode Standard, the "inherited" script property applies to
an individual coded character, such as a combining diacritical mark,
that inherits its script from the preceding character.

5. Reference to published description
of the language (book or article):

6. Any other relevant information:

This subtag corresponds to an ISO 15924 code element for "Code for
inherited script" added 2009-02-23. This code element was added to
address a specific need in the Unicode Standard. A Comment was
added to indicate the intended use of this subtag.
