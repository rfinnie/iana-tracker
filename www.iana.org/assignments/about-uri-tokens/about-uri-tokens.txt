       Token                     Description                                                        Reference
   blank         The about:blank URI references a blank page. [RFC6694]
                 The about:invalid URI references a
                 non-existent document with a generic error
   invalid       condition. It can be used when a URI is      [http://dev.w3.org/csswg/css3-values/#about-invalid][CSSWG]
                 necessary, but the default value shouldn't
                 be resolveable as any type of document.
   srcdoc        Used as the document's address of iframe     [http://www.whatwg.org/specs/web-apps/current-work/#about:srcdoc][Ian_Hickson]
                 srcdoc documents in HTML.
   legacy-compat For use in DOCTYPEs in HTML documents when   [http://www.whatwg.org/specs/web-apps/current-work/#about:legacy-compat][Ian_Hickson]
                 needed for compatibility with XML tools.
